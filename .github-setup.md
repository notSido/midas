# GitHub Setup Instructions

## Step 1: Create the GitHub Repository

1. Go to https://github.com/new
2. Repository name: `midas`
3. Description: `Themida 3.x unpacker for Linux using Unicorn emulation`
4. Visibility: **Public**
5. **DO NOT** initialize with README, .gitignore, or license (we already have these)
6. Click "Create repository"

## Step 2: Push to GitHub

After creating the repo, GitHub will show you commands. Use these instead:

```bash
# Add the remote (replace YOUR_USERNAME with your GitHub username)
git remote add origin https://github.com/YOUR_USERNAME/midas.git

# Push the code
git push -u origin main
```

## Step 3: Verify

Visit: `https://github.com/YOUR_USERNAME/midas`

You should see:
- README.md with project overview
- Complete source code in src/
- SUMMARY.md with detailed documentation
- NOTES.md with development details

## Step 4: Integration

To use in your container project:

```dockerfile
# In your Dockerfile
FROM rust:latest

# Install dependencies
RUN apt-get update && apt-get install -y \
    build-essential \
    libclang-dev \
    cmake \
    && rm -rf /var/lib/apt/lists/*

# Clone and build
RUN git clone https://github.com/YOUR_USERNAME/midas.git /opt/midas
WORKDIR /opt/midas
RUN cargo build --release

# The binary will be at: /opt/midas/target/release/themida-unpack
```

Or as a Git submodule in your existing project:

```bash
cd your-project
git submodule add https://github.com/YOUR_USERNAME/midas.git tools/midas
```

## Troubleshooting

If you get authentication errors:
```bash
# Use SSH instead
git remote set-url origin git@github.com:YOUR_USERNAME/midas.git
git push -u origin main
```

Or use GitHub CLI:
```bash
gh auth login
gh repo create midas --public --source=. --push
```
